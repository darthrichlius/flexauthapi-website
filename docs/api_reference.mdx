#### Credentials (with MFA)

Authenticating with MFA is a 2 steps process. If you want more information about this topic, refers to the workflow reference.

1. <b>Step1</b>, is a "security challenge” step. When successful, provide the
   user with a <code>mfaAccessToken</code>
2. <b>Step2</b>, is a "identity challenge” step. When successful, provides the
   final <code>accessToken</code> JWT needed resource access operations
